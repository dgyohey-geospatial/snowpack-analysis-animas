## Data Sources

This project uses DAYMET Snow Water Equivalent (SWE) data.

Data can be downloaded from:
https://climateengine.org/

Suggested parameters:
- Variable: SWE
- Temporal Resolution: Monthly Mean
- Time Range: 2015–2025
- AOI: Animas River Watershed

Note: Due to file size constraints, raw raster datasets are not included in this repository.
